Introduction

Python is a high-level, interpreted programming language that emphasizes readability and simplicity. It's commonly used in web development, data science, artificial intelligence, and more.

This documentation provides an overview of Python syntax, variables, functions, and control structures like loops.

Python Syntax

Python syntax is clean and easy to learn. Here is a basic "Hello, World!" example:

print("Hello, World!")

Python uses indentation (whitespace) to define blocks of code, instead of curly braces like in other languages.

  • Statements end with a newline, no semicolon required
  • Indentation is used to define code blocks
  • Python is case-sensitive
Variables

Variables in Python are dynamically typed, meaning you don't have to declare their type explicitly. Here's how to declare and assign values:

x = 10
y = "Hello, Python"
z = 3.14
  • Variables do not need a type declaration
  • They are created upon assignment
  • Common data types include int, str, and float
Functions

Functions in Python are blocks of reusable code that can accept parameters and return values. Here's how to define and call a function:

def my_function():
  print("Hello from a function!")

my_function()

Functions can also return values and accept arguments:

def add(a, b):
  return a + b

result = add(5, 3)
print(result) # Outputs 8
Loops in Python

Loops allow you to iterate over a sequence of data (like a list, tuple, or string) and perform repetitive tasks. The two main types of loops in Python are for loops and while loops.

Here's an example of a for loop:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
  print(fruit)

Here's an example of a while loop:

i = 1
while i < 5:
  print(i)
  i += 1
  • for loops iterate over a sequence
  • while loops run as long as a condition is true
  • Both loops can be interrupted using break or continue statements
